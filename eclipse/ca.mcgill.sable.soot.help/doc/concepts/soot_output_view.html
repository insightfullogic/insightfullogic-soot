Soot Output View

This view is where you will see the output generated while running Soot. It behaves much like the Java Console view, but it is much simpler.

You do not need to do anything special to see this view, it is created and given focus as needed. If you accidently close it and want to re-display it. You can do so by selecting Window -> Show View -> Other -> Soot -> Soot Output.

Actions Available

There are two actions available from the context menu in this view.

  • Select All - Selects all text in view.
  • Copy - Copies the selected text in the view.
