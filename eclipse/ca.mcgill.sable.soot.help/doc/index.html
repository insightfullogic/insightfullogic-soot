Soot - Eclipse Plugin


Introduction top

The Soot - Eclipse Plugin integrates Soot a Java optimization framework with Eclipse allowing the user to:

  • optimize class files automatically and use more advanced optimization options
  • import and decompile class files using the Dava Decompiler
  • use a Jimple editor
  • view attribute information at source and IR levels

This plugin works with the Eclipse official release 2.1
Installation Guidelines and Downloads top

Note:This is not an official release of either Soot or the Soot Eclipse Plugin.
Both Soot and the Eclipse Plugin will be changing dramatically in the coming months before the official release.

The first official release of the Soot - Eclipse Plugin will coincide with the official release of Soot version 2.0, due to be released in early June.

Eclipse must already be installed. If you have not installed Eclipse you can get it from www.eclipse.org.

Download this soot-eclipse-plugin.jar (May 22/2003) to the $ECLIPSE_HOME/plugins/ directory and unjar it.
(ex: jar -xvvf soot-eclipse-plugin.jar)
